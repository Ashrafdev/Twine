Requirements
------------

  - [PHP](https://php.net) >= 5.6

Install with Composer
---------------------

```bash
composer require phlak/twine
```

Initializing Twine
------------------

First, import Twine:

```php
use Twine;
```

Then instantiate a Twine string:

```php
$twine = Twine\String('john pinkerton');
```
